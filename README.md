# Filters-based-on-concentrated-elements-with-attenuation-poles
# Проектирование фильтров на сосредоточенных элементах с полюсами затухания на конечных частотах

**Среда разработки:** NI AWR Design Environment (Microwave Office)  
**Типы фильтров:** ФНЧ, ФВЧ (эллиптические, 3-й и 5-й порядки)  
**Статус:** завершён, все характеристики оптимизированы

## Обзор

В рамках проекта спроектированы и оптимизированы:

- ФНЧ 3-го порядка с полюсом затухания на конечной частоте
- ФНЧ 5-го порядка (симметричная и несимметричная схемы)
- ФВЧ 5-го порядка (симметричная и несимметричная схемы)

Для каждого фильтра выполнены:
- расчёт топологии и номиналов LC-элементов
- оптимизация по критериям: КСВН ≤ 1.2 в полосе пропускания, затухание в полосе заграждения ≥ 15…30 дБ
- анализ АЧХ, КСВН, ФЧХ, ГВЗ
- исследование влияния добротности элементов (Q = 1…1000) на потери на частоте среза

## Результаты

Все спроектированные фильтры удовлетворяют техническим требованиям.  
Сводные таблицы характеристик – в `results/Summary_Tables.md`.  
Номиналы элементов – в `circuits/component_values.md`.  
Подробные отчёты по каждому типу – в папке `docs/`.
