# ФНЧ 3-го порядка с полюсом затухания

## Технические требования

| Параметр | Значение |
|----------|----------|
| Частота среза | 100 МГц (опорная) |
| Граничная частота полосы пропускания (fc) | 145 МГц |
| КСВН в полосе пропускания | ≤ 1.2 |
| Затухание в полосе заграждения (f ≥ 1.8·fc) | ≥ 15 дБ |
| Сопротивление нагрузки | 50 Ом |

## Полученные характеристики

| Параметр | Требование | Фактическое значение |

| Частота среза (по уровню -3 дБ) | 145 МГц | 146.4 МГц |

| Неравномерность ГВЗ в ПП | — | 9.18 нс |

### Поведение АЧХ и КСВН

- АЧХ: монотонный спад в полосе заграждения, полюс затухания на конечной частоте (≈ 253 МГц).
- КСВН: не превышает 1.2 вплоть до 145 МГц.

### Групповое время замедления (ГВЗ)

Неравномерность в полосе пропускания составляет 9.18 нс, что допустимо для широкополосных систем.

## Вывод

Фильтр соответствует ТЗ с небольшим отклонением по частоте заграждения (253.6 вместо 261 МГц) и затуханию (14.75 вместо 15 дБ) – в пределах погрешности оптимизации.
