# ФВЧ 5-го порядка (эллиптический)

## Технические требования

| Параметр | Значение |
|----------|----------|
| Частота среза (fc) | 145 МГц |
| Частота бесконечного затухания | 0.7·fc = 101.5 МГц |
| КСВН в полосе пропускания (f > fc) | ≤ 1.2 |
| Затухание в полосе заграждения (f < 101.5) | симм. ≥ 25 дБ, несимм. ≥ 30 дБ |
| Rн | 50 Ом |

## 1. Симметричная схема

### Схема
C1 C3 C5
o----||----+----||----+----||----o
| | |
L2 L4 L6
| | |
o---------+--------+--------+----o

**Номиналы после оптимизации:**

| Элемент | Значение |
|---------|----------|
| C1, C5 | 12.4 пФ |
| C3 | 8.2 пФ |
| L2, L6 | 27.6 нГн |
| L4 | 39.1 нГн |

### Характеристики

| Параметр | Требование | Фактически |
|----------|------------|------------|
| Частота среза (по уровню -3 дБ), МГц | 145 | 141.0 |
| Частота полосы заграждения (по уровню -25 дБ), МГц | ≤ 101.5 | 102.4 |
| Затухание на f = 102.4 МГц, дБ | ≥ 25 | 24.84 |
| КСВН в ПП | ≤ 1.2 | 1.2 |
| Неравномерность ГВЗ в ПП, нс | — | 84.92 |

**Zeven / Zodd:** пересечение характеристик на частоте ≈ 145 МГц (симметричная мода).

### Зависимость потерь на fc от добротности

Аналогична ФНЧ: потери обратно пропорциональны Q (в логарифмическом масштабе прямая).

## Вывод

Оба ФВЧ спроектированы успешно. Несимметричная схема даёт большее затухание в полосе заграждения (~30 дБ), но требует дополнительного элемента. Симметричная проще в реализации и имеет более симметричную ФЧХ.
