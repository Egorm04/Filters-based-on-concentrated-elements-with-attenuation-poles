# ФНЧ 5-го порядка эллиптического типа (симметричная и несимметричная схемы)

## Технические требования

| Параметр | Значение |
|----------|----------|
| Частота среза (fc) | 145 МГц |
| Полоса заграждения | f ≥ 1.3·fc ≈ 188.5 МГц |
| КСВН в ПП | ≤ 1.2 |
| Затухание в ПЗ | симм. схема ≥ 25 дБ, несимм. ≥ 30 дБ |
| Rн | 50 Ом |

**Оптимизированные номиналы:**

| Элемент | Значение |
|---------|----------|
| L1 | 28.4 нГн |
| L3 | 36.7 нГн |

| C4 | 11.5 пФ |
| C5 | 4.1 пФ |

### Характеристики

| Параметр | Требование | Фактически |
|----------|------------|------------|
| Частота среза, МГц | 145 | 144.1 |
| fзаг, МГц | 188.5 | 188.8 |
| Затухание на fзаг, дБ | ≥ 30 | 29.73 |
| КСВН в ПП | ≤ 1.2 | 1.2 |
| Неравномерность ГВЗ, нс | — | 6.55 |

**Примечание:** несимметричная схема имеет **два полюса затухания** (≈ 195 МГц и 225 МГц), что обеспечивает более крутой спад.

### Зависимость потерь от Q

Аналогична симметричной схеме (потери удваиваются при уменьшении Q вдвое).

## Вывод

Обе схемы соответствуют ТЗ. Несимметричная схема обеспечивает более глубокое затухание (29.7 дБ против 24.8 дБ) ценой асимметрии АЧХ и незначительного увеличения ГВЗ.
